News API

About

News API was built to present my natural language processing (NLP) classifier to categorise news under different topics. It does not depend upon any third party API, rather it scrapes news headlines from different website.

Documentation

Endpoint /news/:query
Parameters query: (required) search for particular news
Response JSON object with fields:
title
category
link
